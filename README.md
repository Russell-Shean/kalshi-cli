# kalshi-cli
This is a python cli that collects and cleans sports data from Kalshi and google sheets

# Setup
You will need to create two sets of credentials for this project: Kalshi and Google. Kalshi is very straightforward to setup and google is somewhat complicated. If you don't making the spreadsheet publically visible the google credentials setup steps can be completely skipped.
